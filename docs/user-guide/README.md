# 天文 V2 POS 使用手冊

> 版本：v2.10.0 | 最後更新：2026-04-11

歡迎使用天文精緻便當 POS 系統。本手冊將協助您快速上手日常操作，從點餐、打卡到管理功能一應俱全。

---

## 五分鐘快速上手

以下是每天最常用的 5 個操作，幫助您立即開始使用：

| # | 操作 | 說明 | 詳細教學 |
|---|------|------|----------|
| 1 | 點餐下單 | 點選商品、調整數量、送出訂單 | [點餐操作](10-點餐操作.md#瀏覽商品與分類) |
| 2 | 打卡上班 | 點擊自己的名字，確認打卡 | [打卡與出勤](20-打卡與出勤.md#打卡上班) |
| 3 | 修改訂單 | 在近期訂單中左滑編輯 | [點餐操作](10-點餐操作.md#編輯已送出訂單) |
| 4 | 查看營收 | 進入「統計」查看今日銷售數據 | [管理功能](30-管理功能.md#營收統計) |
| 5 | 手動備份 | 設定 → 雲端備份 → 立即備份 | [雲端備份](40-雲端備份.md#手動備份) |

## 如何使用本手冊

- 如果您是第一次使用天文 V2，請從 [首次設定](00-首次設定.md) 開始閱讀
- 如果您已經在使用，可以直接跳到需要的章節
- 每個章節末尾都有「小提醒」和「常見問題」，遇到問題時可以先查看
- 截圖中的畫面可能因版本更新而略有不同，但操作流程基本一致
- 標示「需管理員權限」的功能，需要使用 Google 帳號登入後才能操作

---

## 裝置需求

- iPad 10（2022）或 iPad 11（2025）以上機型
- Safari 瀏覽器
- 網路連線（首次安裝及雲端備份時需要）
- 應用程式僅支援直向（直式）模式
